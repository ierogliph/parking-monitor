# 🅿️ Мониторинг парковки ЖК "Первый Московский"

Telegram-бот для автоматического отслеживания стоимости и наличия парковочных мест на сайте [absrealty.ru](https://www.absrealty.ru).
Этот бот поможет вам быть в курсе изменений, чтобы вы могли принять решение об аренде или покупке парковочного места вовремя.
## ✨ Возможности

*   🔄 **Автоматический мониторинг**: Проверяет сайт каждые 5 минут (настраивается).
*   🔔 **Уведомления**: Мгновенно сообщает в Telegram об изменении цены или количества мест.
*   👥 **Подписки**: Пользователи могут подписываться на рассылку уведомлений.
*   📊 **Статистика**: Админ видит количество подписчиков и лог действий.
*   🐳 **Docker**: Легкий запуск через Docker Compose.

## 🚀 Установка и запуск

### Предварительные требования
*   Docker и Docker Compose
*   Токен Telegram бота (от @BotFather)

### Инструкция

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/your-username/parking-monitor.git
    cd parking-monitor
    ```

2.  **Настройте окружение:**
    Создайте файл `.env` на основе примера:
    ```bash
    cp .env.example .env
    ```
    Откройте `.env` и вставьте свой токен бота и ваш ID (чтобы быть админом).

3.  **Запустите контейнер:**
    ```bash
    docker-compose up -d --build
    ```

## 🛠 Команды бота

*   `/start` — Запуск и приветствие.
*   `/info` — Текущий статус (цена и места).
*   `/subscribe` — Подписаться на уведомления об изменениях.
*   `/unsubscribe` — Отписаться.
*   `/help` — Справка.

### Команды администратора
*   `/stats` — Показать статистику подписчиков и лог.
*   `/simulate` — Симуляция изменения данных (для проверки уведомлений).
*   `/test` — Отправка тестового сообщения (проверка верстки).

## 📂 Структура проекта

*   `parking_monitor.py` — Основной код бота.
*   `data/` — Папка с БД (SQLite) и логами (создается автоматически, сохраняется при перезапуске).
